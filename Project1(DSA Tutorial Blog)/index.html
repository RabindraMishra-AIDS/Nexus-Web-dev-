Introduction to C++

C++ is powerfull general purpose programming language that was developed as an extension of c programming language by Bjarne Stroustrup in the early 1980's.C++ is widely used for system/software development,game development and applications that require high performance.

Features of C++

  • Object-Orineted:C++ supports object oriented programming (OPP's) Principles. Thus allowing us to create classes and objects.
  • Rich Standard Libraries:C++ Provides variety of in built functions and Libraries,which can significantly reduce development time.
  • Compatibility: C++ is compatible with C,allowing for integration in various codes
  • Performance: C++ allows low level memory manipulation thus allowing developers to build optimized applications.

Introduction to Arrays

What is an Array?

An array is a datastructure that stores the element in a continous memory block.But the elements must be of the same datatypes. Basically it helps us to store multiple elements inside a single variable allowing efficient data management and manipulation.



Characteristics of Array

  • Fixed Size: Size oof an array is always defined when created and cannot be changed during runtime
  • Indexed:Each elements in an array is indexed so that it can be accesed easily.
  • Homogenous Elements: Each elements of an array are of same datatypes.

Types of Arrays

  1. One-Dimensional Arrays:Linear List of elements where each element can be acessed using single index.
    • Example: int num[5]; (stores 5 Elements).
  2. Multi-Dimensional Arrays:Arryas with more than one dimensions commonly used for Matrices or tables.
    • Example: int num[3][3]; (a 3x3 Matrix).
  3. Dynamic Arrays:Arrays that can change size during Program execution typically implemented using data structures like lists.

Introduction To Trees

Tree Data Structure is a non-linear data structure in which a collection of elements known as nodes are connected to each other via edges such that there exists exactly one path between any two nodes.


Chatacteristics of Tree

  • Root Node: Node that does not has any precedend.
  • Parent Node:A node that has one or more children.
  • Child Node:A node that is descendent of another Node.
  • Leaf Node: A node that does not have any children

Types of Tree

  1. Binary Search Tree
  2. AVL Tree
  3. Red Black Tree
  4. N-array Tree
